# SyncNode-Backend

> **Краткое описание проекта:** Серверная(Backend) часть проекта SyncNode. Предполагается мгновенная синхронизация заметок в формате Markdown. Frontend реализован будет как плагин на Obsidian(отдельный репозиторий). Так же хочется реализовать многопользовательскую синхронизацию заметок(а именно редактирование заметок вместе в реальном времени).

## 🛠 Техно-стек

- Язык: `Python 3.13`.
- Пакетный менеджер: `uv` (`hatchling` под капотом).
- Тестирование: `Pytest` + `Mock`.
- База данных: `PostgreSQL` + `SQLAlchemy`.
- Миграции: `Alembic`.
- Брокер сообщений и кэширование: `Redis`.
- Валидация данных: `Pydantic v2`.
- Веб-фреймворк: `FastAPI` + `Uvicorn`.
- Синхронизационный движок: `py-crdt`.
- Инфраструктура: `Docker` & `Docker Compose`.

## 🗄 Структура проекта

### Структура бд

#### Таблицы

| Таблица             | Описание                                                                                                                                   |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| users               | Содержит информацию о учётных записях (JWT).                                                                                               |
| notes               | Содержит актуальную информацию о заметках. Метаданные файла и его актуальное бинарное состояние.                                           |
| notes_collaborators | Связующая таблица для организации доступа (многие-ко-многим). Если записи в этой таблице нет, то доступ к заметке имеет только её `owner`. |

##### Users

| Поле            | Тип          | Ограничения                             | Описание                                |
| --------------- | ------------ | --------------------------------------- | --------------------------------------- |
| id              | UUID         | Primary Key, default=uuid_generate_v4() | Уникальный идентификатор пользователя.  |
| name            | VARCHAR(255) | Unique, Not Null                        | Имя пользователя для входа.             |
| hashed_password | VARCHAR(255) | Not Null                                | Хеш пароля пользователя.(Bcrypt/Argon2) |
| created_at      | TIMESTAMP    | Not Null, default=now()                 | Дата регистрации/создания пользователя. |

##### Notes

| Поле       | Тип           | Ограничения                                | Описание                                                   |
| ---------- | ------------- | ------------------------------------------ | ---------------------------------------------------------- |
| id         | UUID          | Primary Key                                | Уникальный идентификатор заметки(Генерируется на сервере). |
| owner_id   | UUID          | Foreign Key -> users.id, On Delete Cascade | ID владельца заметки.                                      |
| path       | VARCHAR(1024) | Not Null                                   | Относительный путь в Vault (например, Work/todo.md).       |
| crdt_state | BYTEA         | Nullable                                   | Бинарный слепок документа (Yjs / py-crdt).                 |
| updated_at | TIMESTAMP     | Not Null, default=now()                    | Последнее сохранение.                                      |

##### Notes_collaborators

| Поле         | Тип         | Ограничения                                   | Описание                                |
| ------------ | ----------- | --------------------------------------------- | --------------------------------------- |
| id           | UUID        | Primary Key                                   | Уникальный ID связи.                    |
| note_id      | UUID        | Foreign Key -> notes.id, On Delete Cascade    | Ссылка на документ.                     |
| user_id      | UUID        | Foreign Key -> users.id, On Delete Cascade    | Ссылка на приглашенного соавтора.       |
| access_level | VARCHAR(20) | Not Null (Проверка: 'read', 'write', 'admin') | Роль: только чтение или редактирование. |
